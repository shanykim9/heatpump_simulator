# 🏡 스마트 히트펌프 에너지 및 경제성 시뮬레이터

**2026년 전기요금 개편안을 완벽 반영한 히트펌프 에너지 시뮬레이션 & 전기요금 경제성 분석 웹앱**

---

## 주요 기능

### 1. 열역학 물리 시뮬레이션
- **히트펌프 성능 2D 보간**: 외기온도(-25~43℃)와 출수온도(35~75℃)에 따른 난방 능력(Capacity) 및 효율(COP)을 `RegularGridInterpolator`로 정밀 계산
- **제상(Defrost) 손실 로직**: 외기온도 5℃ 미만에서 선형 비례 감소 반영 (최소 80% 제한)
- **실내 열평형 모델**: 주택 단열(UA 계수), 열용량, 바닥 방열계수(한국지역난방공사 기준)를 기반으로 분 단위 온도 계산
- **직수(급수) 온도 계절 모델**: 연간 사인파 모형으로 상수도 온도 변화 추정

### 2. 스마트 제어 알고리즘
- **히스테리시스 제어**: 잦은 ON/OFF 방지를 위한 재가동 온도차 설정
- **3-Way 밸브 (급탕 우선) 모드**: 온수 사용 중 난방 일시 중단
- **요금제 맞춤형 스마트 제어**:
  - 봄/가을 주말·공휴일 11~14시: 할인 시간대 이용한 선행 축열
  - 18~21시 최대부하(피크) 시간대: 히트펌프 가동 자동 회피
- **법정 공휴일 자동 인식** (`holidays` 라이브러리 연동)

### 3. 외부 API 연계
| API | 용도 | 비고 |
|---|---|---|
| [Open-Meteo](https://open-meteo.com/) | 실측 기상 데이터 (시간별 외기온도) | 무료, 키 불필요 |
| [NASA POWER](https://power.larc.nasa.gov/) | 태양광 일사량 데이터 | 무료, 키 불필요 |
| 한국전력거래소 SMP API | 계통한계가격(SMP) 자동 조회 | 공공데이터포털 키 사용 |
| 서부발전 REC API | 신재생에너지 공급인증서 단가 | 공공데이터포털 키 사용 |

### 4. 전기요금 계산 (2026년 개편안 반영)
| 요금제 | 특징 |
|---|---|
| **일반용 전력(갑) I** | 경부하 단가 인상, 최대부하(18~21시) 단가 인하, **봄/가을 주말 11~14시 50% 반값 할인** |
| **주택용 전력(누진제)** | 여름(7~8월) / 기타 계절 구간 분리, 1000kWh 초과 할증 적용 |
| **주택용 계절·시간대별(TOU)** | 시간대별 차등 단가 (경부하/중간부하/최대부하) |

### 5. 태양광(PV) 연동
- NASA POWER API 기반 실제 일사량 적용 (API 실패 시 표준 맑은 날 곡선 자동 대체)
- 자가소비량과 잉여 매전량 분리 계산
- SMP + REC 단가 기반 월 예상 매전수익 산출

## 입력 파라미터 가이드

### 지역 및 기간
- **시/도, 시/군/구**: 전국 모든 행정구역 지원 (지역계수 자동 산정)
- **시작일 / 종료일**: 과거 날짜 입력 시 Open-Meteo 실측 기상 데이터 사용

### 주택 설비 정보
| 항목 | 설명 |
|---|---|
| 건축년도 | 단열 성능 계수 자동 적용 (1.4 / 0.7 / 0.35) |
| 주택형태 | 아파트(노출면적 40%) / 단독주택 |
| 전용면적 | 바닥 방열계수 및 UA값 계산에 활용 |
| 지역계수 | 제주(0.7) / 남부(0.85) / 중부2(1.0) / 중부1(1.1) |
| 위치 | 중간층 / 최상층 / 최하층 (추가 노출면 반영) |

### 히트펌프 운전 설정
| 항목 | 기본값 | 설명 |
|---|---|---|
| 물탱크 용량 | 300 L | 100~600 L |
| 기본 목표온도 | 50 ℃ | 40~75 ℃ |
| 재가동 온도차 | 5 ℃ | 히스테리시스 폭 |
| 실내 목표온도 | 23 ℃ | 18~30 ℃ |
| 난방 가동 시간 | 22~06시 | 멀티셀렉트로 선택 |

### 온수 스케줄
- 샤워, 세면, 설거지, 세탁기 4종
- 평일 / 주말·공휴일 패턴 별도 설정 가능
- 건당 시작 시각, 지속 시간 설정, 복수 일정 추가 가능
